# Release Notes Template

## Version 1.0 - Initial Release

### What is Pet Scheduling?
Pet Scheduling is an app to help you manage your pets' care schedules, track tasks, and keep important pet information organized.

### Key Features to Test:
- ✅ **Pet Management**: Add, edit, and delete pets with photos
- ✅ **Task Scheduling**: Create recurring tasks (feeding, medication, grooming, etc.)
- ✅ **Notifications**: Get reminders for scheduled tasks
- ✅ **Cloud Sync**: Your data syncs across devices via Firebase
- ✅ **Google Sign-In**: Easy authentication with your Google account
- ✅ **Dark Theme**: Toggle between light and dark themes
- ✅ **Emergency Contacts**: Store veterinarian and emergency contact information
- ✅ **Family Sharing**: Share pets with family members (coming soon)
- ✅ **Statistics**: View completion rates and task analytics
- ✅ **Export**: Export schedules and pet details for vet records

### What to Test:
1. **Sign in with Google** - Make sure authentication works
2. **Add a pet** - Test adding pets with photos
3. **Create tasks** - Schedule recurring tasks and set reminders
4. **Notifications** - Verify you receive task reminders
5. **Cloud sync** - Add data on one device, check if it syncs
6. **Dark theme** - Toggle themes and check visibility
7. **Delete pets** - Test deletion functionality

### Known Issues:
- None currently - please report any bugs you find!
